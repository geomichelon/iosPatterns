# PatternsIOS

Design Patterns study project for iOS with Swift and SwiftUI examples, unit tests, and HTML tutorials.

## Contents

1. Swift and SwiftUI examples organized by pattern.
2. Unit tests for each pattern.
3. Bilingual HTML tutorials (EN/PT) with diagrams and SOLID mapping.

## How to run

1. Generate the project: `tuist generate`
2. Open `PatternsIOS.xcworkspace`
3. Run the `PatternsIOSApp` scheme

## HTML tutorials

- English: `Sources/PatternsIOS/Patterns/tutorials.html`
- Portuguese: `Sources/PatternsIOS/Patterns/tutorials-pt.html`

## Structure

- `Sources/PatternsIOS/Patterns/<Pattern>/Swift/`
- `Sources/PatternsIOS/Patterns/<Pattern>/SwiftUI/`
- `Sources/PatternsIOS/Patterns/<Pattern>/tutorial.html`
- `Sources/PatternsIOS/Patterns/<Pattern>/tutorial-pt.html`
- `Tests/PatternsIOSTests/Patterns/`
